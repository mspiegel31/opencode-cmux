```typescript
export { CmuxPlugin } from "./cmux-notify.js"

```

Critical notes:
- Use `.js` file extensions in the import paths (NOT `.ts`). With `"moduleResolution": "bundler"` and `"verbatimModuleSyntax": true`, TypeScript emits the import paths as-is into dist/index.js. ESM requires explicit `.js` extensions for relative imports.
- Use named re-exports (not `export * from`) — explicit exports make the public API clear.
- No default export — OpenCode plugin loader expects named exports.
  </action>
